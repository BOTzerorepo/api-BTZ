Autenticación

La API utiliza JWT (JSON Web Tokens) para la autenticación. Incluye el token en el header de tus requests:

Authorization: Bearer {tu_token_jwt}

Ejemplo de Login

const response = await fetch('https://tu-dominio.com/api/login', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    email: 'usuario@ejemplo.com',
    pass: 'tu_contraseña'
  })
});

const data = await response.json();
console.log(data.token); // Guarda este token para futuras requests
